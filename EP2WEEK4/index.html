Introduction

Stack กับ Queue คืออะไร?

ทั้ง Stack และ Queue เป็น Data Structure เชิงเส้น ที่ควบคุมว่าข้อมูลเข้า-ออกได้อย่างไร ต่างกันตรงที่ กฎการเข้า-ออก

📚
Stack — LIFO
Last In, First Out
ตัวที่เข้าหลังสุด ออกก่อน — เหมือนกองจาน
ตัวอย่าง: กองจาน, Undo/Redo, browser back, recursion call stack
🚶
Queue — FIFO
First In, First Out
ตัวที่เข้าก่อน ออกก่อน — เหมือนคิวรอ
ตัวอย่าง: คิวรอ, print queue, BFS, task scheduler
Push/Enqueue
O(1)
เพิ่มข้อมูลเข้า
Pop/Dequeue
O(1)
นำข้อมูลออก
Peek/Front
O(1)
ดูข้อมูลบนสุด/หน้าสุด
ทั้ง Stack และ Queue มีความซับซ้อน O(1) ทุก operation ทำให้เป็นที่นิยมในการแก้โจทย์แข่งขัน เช่น TOI, สอวน.

Key Takeaways

สถานการณ์ใช้อะไรเพราะ
ต้องการ LIFO (ล่าสุดออกก่อน)Stack (list)append/pop = O(1)
ต้องการ FIFO (แรกเข้าออกก่อน)dequepopleft() = O(1)
ตรวจวงเล็บสมดุลStackเปิด=push, ปิด=pop เปรียบเทียบ
หา Next Greater ElementMonotonic StackO(N) แทน O(N²)
