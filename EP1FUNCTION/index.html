Function คืออะไร?

Function คือ "กล่องคำสั่ง" ที่เราสร้างขึ้นเพื่อเก็บชุดคำสั่งที่ใช้บ่อยๆ เหมือน บล็อก LEGO ที่ต่อขึ้นมาเองแล้วสามารถนำมาใช้ซ้ำกี่ครั้งก็ได้โดยไม่ต้องต่อใหม่ทุกครั้ง

🧱
def — นิยามฟังก์ชั่น
ใช้คำสั่ง def เพื่อ "สร้าง" บล็อก LEGO ตั้งชื่อและกำหนดว่าทำอะไร
📞
เรียกใช้งาน
พิมพ์ชื่อฟังก์ชั่นพร้อม () เพื่อ "เรียกใช้" บล็อก LEGO ได้กี่ครั้งก็ได้
📦
Arguments — รับข้อมูล
ส่งข้อมูลเข้าไปในฟังก์ชั่นเพื่อให้ทำงานกับค่าที่แตกต่างกันได้
🎁
Return — ส่งผลลัพธ์กลับ
ใช้ return เพื่อส่งผลลัพธ์ออกมาจากฟังก์ชั่นให้ใช้งานต่อได้
① def function_name():
→
② เรียกใช้ function_name()
→
③ ทำงานภายใน
→
④ return ผลลัพธ์

Python มีกฎในการตั้งชื่อฟังก์ชั่นเพื่อให้โปรแกรมทำงานถูกต้องและอ่านเข้าใจง่าย

3Case-sensitive — ต่างกันหมายถึงคนละฟังก์ชั่นGreet() ≠ greet()

Part 2

Arguments — รับค่าเข้าฟังก์ชั่น

Arguments คือตัวแปรที่กำหนดไว้ในวงเล็บของฟังก์ชั่น เพื่อรับค่าที่ส่งเข้ามาและนำไปใช้ภายใน

รูปแบบ: def my_function(arg1, arg2): — คั่นด้วยจุลภาค (,) เพิ่มได้ไม่จำกัด
Positionalตามลำดับ
ส่งค่าตามลำดับที่กำหนด greet("Alice", 25)
Keywordระบุชื่อ
ระบุชื่อ argument เพื่อกำหนดค่า greet(name="Bob")

Default Arguments — ค่าเริ่มต้น

กำหนดค่าเริ่มต้นให้ argument เพื่อไม่ต้องส่งค่าเดิมซ้ำๆ ทุกครั้งที่เรียกใช้

⚠️ กฎ: Default argument ต้องอยู่ หลัง argument ที่ไม่มีค่าเริ่มต้นเสมอ — def f(a, b=5) ✅ แต่ def f(a=5, b) ❌

Part 3

Return — ส่งผลลัพธ์กลับ

ใช้ return เพื่อส่งค่าออกจากฟังก์ชั่น ทำให้สามารถนำผลลัพธ์ไปใช้งานต่อหรือเก็บในตัวแปรได้

✅ ข้อดีของ return: ฟังก์ชั่นกลายเป็น "ค่า" ที่สามารถใส่ในนิพจน์, เก็บในตัวแปร, หรือส่งต่อเป็น argument ฟังก์ชั่นอื่นได้
⚠️ หมายเหตุ: หลังจาก return โปรแกรมออกจากฟังก์ชั่นทันที โค้ดที่อยู่หลัง return จะไม่ถูกรัน
