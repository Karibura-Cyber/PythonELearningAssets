😱 ทำไม Recursion ธรรมดาถึงช้า?

ลองนึกถึง Fibonacci — การหา fib(5) ต้องไปเรียก fib(3) ซ้ำกันถึง 2 ครั้ง! ถ้าหา fib(50) คอมพิวเตอร์จะทำงานซ้ำเป็นล้านล้านรอบ (เครื่องค้างแน่นอน)

😭 Recursion เปล่า O(2ⁿ)
fib(5) เรียก fib(4) และ fib(3)
fib(4) เรียก fib(3) และ fib(2)
fib(3) ถูกคำนวณซ้ำ 2 ครั้ง!
fib(50) → คำนวณซ้ำ ~250 ครั้ง = เครื่องค้าง!
🚀 DP + Memoization O(n)
คำนวณ fib(3) ครั้งแรก → จดไว้ใน memo
ครั้งต่อไปที่ต้องการ fib(3) → เปิดสมุดดูเลย!
fib(50) → คำนวณแค่ 50 ครั้ง = ทันที!
ประหยัดเวลาได้มหาศาล
📊 เปรียบเทียบ: fib(50) แบบ Recursion เปล่า ต้องคำนวณ ~2,251,799,813,685,248 ครั้ง แต่แบบ DP จำเป็นแค่ 50 ครั้ง = เร็วกว่า 45 ล้านล้านเท่า!

Fibonacci DP

1

fib_dp(n) — หา Fibonacci ด้วย Memoization

ใช้ memo = {} เพื่อหลีกเลี่ยงการคำนวณซ้ำ · F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2)

ตัวอย่าง call
fib_dp(0) fib_dp(1) fib_dp(10) fib_dp(20)
ผลลัพธ์
0 1 55 6765
เขียนโค้ดex1.py
memo = {}
def fib_dp(n):
  if n in memo: return memo[n]
  if n <= 1: return n
  memo[n] = fib_dp(n-1) + fib_dp(n-2)
  return memo[n]

def climb_stairs(n):

Min Cost Climbing Stairs

3

min_cost_stairs(cost) — ขึ้นบันไดด้วยพลังงานน้อยสุด

cost[i] คือพลังงานที่ขั้น i · เริ่มที่ขั้น 0 หรือ 1 · สูตร: min_cost(i) = cost[i] + min(min_cost(i+1), min_cost(i+2))

ตัวอย่าง call

ผลลัพธ์
15 6 0
เขียนโค้ดex3.py
def min_cost_stairs(cost):
  memo = {}
  def dp(i):
    if i >= len(cost): return 0
    if i in memo: return memo[i]
    memo[i] = cost[i] + min(dp(i+1), dp(i+2))
    return memo[i]
  return min(dp(0), dp(1))

Grid Traveler

4

grid_traveler(m, n) — หาจำนวนเส้นทางในตาราง m×n

เดินได้แค่ขวาหรือลง · Base: 1×1=1, 0×n=0 · key ของ memo คือ tuple (m,n)

ตัวอย่าง call
grid_traveler(1,1) grid_traveler(2,3) grid_traveler(3,3) grid_traveler(5,5)
ผลลัพธ์
1 3 6 70
เขียนโค้ดex4.py
memo = {}
def grid_traveler(m, n):
  if (m,n) in memo: return memo[(m,n)]
  if m==1 and n==1: return 1
  if m==0 or n==0: return 0
  memo[(m,n)] = grid_traveler(m-1,n) + grid_traveler(m,n-1)
  return memo[(m,n)]

House Robber — สายลับขโมยทอง

5

house_robber(gold) — หาทองสูงสุดที่ขโมยได้โดยไม่ปล้นบ้านติดกัน

max( gold[i] + rob(i-2), rob(i-1) ) · Base: rob(-1)=0, rob(0)=gold[0]

ตัวอย่าง call
house_robber([1,2,3,1]) house_robber([2,7,9,3,1]) house_robber([5,1,1,5])
ผลลัพธ์
4 12 10
เขียนโค้ดex5.py
def house_robber(gold):
  memo = {}
  def rob(i):
    if i < 0: return 0
    if i == 0: return gold[0]
    if i in memo: return memo[i]
    memo[i] = max(gold[i]+rob(i-2), rob(i-1))
    return memo[i]
  return rob(len(gold)-1)

Count Ways (k-step)

6

count_ways(n, k) — นับวิธีขึ้นบันไดที่ก้าวได้ 1 ถึง k ขั้น

เหมือน Climbing Stairs แต่ก้าวได้สูงสุด k ขั้น · ways(n) = ways(n-1) + ways(n-2) + ... + ways(n-k)

ตัวอย่าง call

ผลลัพธ์
3 7 8 13
เขียนโค้ดex6.py
def count_ways(n, k):
  memo = {}
  def dp(i):
    if i == 0: return 1
    if i < 0: return 0
    if i in memo: return memo[i]
    memo[i] = sum(dp(i-s) for s in range(1,k+1))
    return memo[i]
  return dp(n)

Unique Paths with Obstacle

8

unique_paths_obs(grid) — Grid Traveler ที่มีสิ่งกีดขวาง

grid[i][j] == 1 คือสิ่งกีดขวาง (ผ่านไม่ได้) · 0 คือทางเปิด · เดินจาก [0][0] ไป [m-1][n-1]

ตัวอย่าง call
unique_paths_obs([[0,0,0],[0,1,0],[0,0,0]]) unique_paths_obs([[0,1],[0,0]]) unique_paths_obs([[1,0]])
ผลลัพธ์
2 1 0
เขียนโค้ดex8.py
def unique_paths_obs(grid):
  m, n = len(grid), len(grid[0])
  memo = {}
  def dp(r, c):
    if r >= m or c >= n or grid[r][c]==1: return 0
    if r==m-1 and c==n-1: return 1
    if (r,c) in memo: return memo[(r,c)]
    memo[(r,c)] = dp(r+1,c) + dp(r,c+1)
    return memo[(r,c)]
  return dp(0,0)

Coin Change DP — จำนวนเหรียญน้อยสุด (ทุกชุดเหรียญ)

9

coin_change_dp(coins, amount) — หาจำนวนเหรียญน้อยสุดที่ถูกต้องเสมอ

DP แก้จุดอ่อนของ Greedy ที่ล้มเหลวกับเหรียญแปลกๆ · dp(amount) = 1 + min(dp(amount-c) for c in coins)

ตัวอย่าง call
coin_change_dp([1,3,4], 6) coin_change_dp([1,5,6,9], 11) coin_change_dp([2], 3)
ผลลัพธ์
2 2 -1
💡 ตัวอย่าง: [1,3,4] ทอน 6 — Greedy เลือก 4+1+1=3 เหรียญ แต่ DP รู้ว่า 3+3=2 เหรียญดีกว่า! คืน -1 ถ้าทอนไม่ได้
เขียนโค้ดex9.py
def coin_change_dp(coins, amount):
  memo = {}
  def dp(rem):
    if rem == 0: return 0
    if rem < 0: return float('inf')
    if rem in memo: return memo[rem]
    best = min(1+dp(rem-c) for c in coins)
    memo[rem] = best
    return best
  res = dp(amount)
  return res if res != float('inf') else -1

Longest Increasing Subsequence

10

lis(nums) — หาความยาวของลำดับย่อยที่เพิ่มขึ้นยาวสุด

ไม่ต้องเลือกตัวเลขติดกัน แต่ต้องเพิ่มขึ้นตลอด · dp[i] = 1 + max(dp[j] for j < i if nums[j] < nums[i])

ตัวอย่าง call
lis([10,9,2,5,3,7,101,18]) lis([0,1,0,3,2,3]) lis([7,7,7,7,7])
ผลลัพธ์
4 4 1
เขียนโค้ดex10.py
def lis(nums):
  n = len(nums)
  dp = [1] * n
  for i in range(1, n):
    for j in range(i):
      if nums[j] < nums[i]:
        dp[i] = max(dp[i], dp[j]+1)
  return max(dp)
