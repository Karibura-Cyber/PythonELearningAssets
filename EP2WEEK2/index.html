Operation Complexity Notes
list[i]O(1)Direct array access
list[i] = xO(1)Direct index assignment
append(x)O(1)*Amortized — occasional O(n) resize
pop()O(1)Remove last element
pop(i)O(n)Shift elements after i
insert(0, x)O(n)Shift all elements right
insert(i, x)O(n)Shift elements from i onward
remove(x)O(n)Search then shift
x in listO(n)Linear search
len(list)O(1)Cached length attribute
list[a:b]O(k)k = b - a elements copied
list + list2O(n+m)Copy both lists
sorted(list)O(n log n)Timsort — stable sort
list.sort()O(n log n)In-place Timsort
min(list) / max(list)O(n)Full scan required
list.reverse()O(n)In-place reversal
list.count(x)O(n)Full scan
list.index(x)O(n)Linear search
list.copy()O(n)Shallow copy
del list[i]O(n)Shift elements after i
