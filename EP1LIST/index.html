List คืออะไร?

List คือโครงสร้างข้อมูล (Data Structure) แบบหนึ่งใน Python ที่ใช้สำหรับเก็บข้อมูลหลายค่าไว้ในตัวแปรเดียว โดยสามารถเก็บข้อมูลได้หลากหลายประเภท ทั้ง int, str, float, bool หรือแม้แต่ List ซ้อน List

📦
Mutable (เปลี่ยนได้)
สามารถแก้ไข เพิ่ม ลบข้อมูลได้หลังสร้างแล้ว ต่างจาก Tuple ที่เปลี่ยนไม่ได้
fruits[0] = "mango" # ได้เลย!
🔢
Ordered & Indexed
ข้อมูลเรียงตามลำดับ เข้าถึงได้ด้วย Index เริ่มต้นที่ 0 และนับถอยหลังได้ด้วย -1
fruits[0], fruits[-1]
🔄
Mixed Types
เก็บข้อมูลต่างชนิดกันได้ในรายการเดียว เช่น ตัวเลข, ข้อความ, บูลีน
mixed = [1, "hello", 3.14, True]
🪆
Nested List
ซ้อน List ใน List ได้ ใช้สร้าง Matrix หรือตารางข้อมูล 2 มิติ
matrix = [[1,2],[3,4],[5,6]]

Slicing — ตัดข้อมูลเป็นช่วง

ใช้ list[start:stop:step] เพื่อดึงข้อมูลบางส่วน โดย stop ไม่รวมตำแหน่งนั้น

Syntaxความหมายตัวอย่าง (fruits = ["apple","banana","cherry","date","elderberry"])
list[1:3]index 1 ถึง 2["banana","cherry"]
list[:2]ตั้งแต่ต้น ถึง index 1["apple","banana"]
list[2:]ตั้งแต่ index 2 ถึงท้าย["cherry","date","elderberry"]
list[::2]ทุกๆ 2 ตัว["apple","cherry","elderberry"]
list[::-1]กลับหน้าหลัง["elderberry","date","cherry","banana","apple"]
