Dictionary เป็นโครงสร้างข้อมูลที่ทรงพลังที่สุดใน Python — ค้นหาข้อมูลด้วย key แทนที่จะใช้ตำแหน่ง index เหมือน List เหมาะสำหรับเก็บข้อมูลนักเรียน, สินค้า, สมาชิก และทุกอย่างที่มีชื่อ

บทนำ

Dictionary คืออะไร?

ลองนึกถึง พจนานุกรมภาษาอังกฤษ — คุณค้นหาคำ (key) แล้วเจอความหมาย (value) นั่นคือหลักการของ Dictionary ใน Python

Dictionary ต่างจาก List ตรงที่ List ใช้ตำแหน่ง [0], [1], [2] แต่ Dictionary ใช้ ชื่อที่มีความหมาย เช่น ["name"], ["age"], ["score"]

📋
List — ใช้ตำแหน่ง
เข้าถึงด้วย index เช่น student[0] ไม่รู้ว่าคือข้อมูลอะไร
student = ["Alice", 15, "A"]
student[0] → "Alice"
🗂
Dictionary — ใช้ key
เข้าถึงด้วยชื่อ เช่น student["name"] อ่านง่าย เข้าใจทันที
student = {"name":"Alice"}
student["name"] → "Alice"

คุณสมบัติสำคัญของ Dictionary

คุณสมบัติรายละเอียดตัวอย่าง
key-value pairทุกข้อมูลมี key และ value คู่กัน"name": "Alice"
Key ห้ามซ้ำkey แต่ละตัวต้องไม่เหมือนกัน (ถ้าซ้ำจะทับกัน)❌ "age":15, "age":20
Key ต้อง immutablekey ต้องเป็น string, int, tuple (ไม่ใช่ list)✓ "name", 1, (1,2)
Value ใดก็ได้value เป็นอะไรก็ได้ รวมถึง list, dict ซ้อน"scores": [90, 85, 95]
ค้นหาเร็ว O(1)ไม่ว่ามีข้อมูลกี่ล้าน ค้นหาได้เร็วเท่ากันhash table ภายใน
Python 3.7+ เรียงลำดับรักษาลำดับการเพิ่มข้อมูล (insertion order)วนซ้ำได้ตามลำดับ

การเข้าถึงข้อมูล (Access)

มี 2 วิธีหลักในการดึงค่าจาก Dictionary โดยแต่ละวิธีมีพฤติกรรมต่างกันเมื่อ key ไม่มีอยู่

⚠️ วิธีที่ 1 — Bracket [ ]

ถ้า key ไม่มี → เกิด KeyError โปรแกรมหยุด!

person["name"] # ✓ ปกติ
person["height"] # ❌ KeyError
✅ วิธีที่ 2 — .get(key, default)

ถ้า key ไม่มี → คืนค่า default (ไม่ error) แนะนำใช้วิธีนี้

person.get("name") # ✓ คืนค่า
person.get("height", 0) # ✓ คืน 0

Safe Access วิธีต่างๆ

มีหลายเทคนิคในการ access ข้อมูลอย่างปลอดภัย ขึ้นอยู่กับสถานการณ์

in operator ตรวจสอบ
เช็คว่า key มีอยู่หรือไม่ก่อนดึงค่า
"name" in person # True/False
try / except จัดการ error
พยายามดึงค่า ถ้า error ก็จัดการแทน
try: x = d["k"]
except KeyError: x = 0
setdefault() เพิ่มถ้าไม่มี
ถ้า key ไม่มี → สร้างให้พร้อมค่า default
d.setdefault("items", [])
Nested .get() dict ซ้อน
เข้าถึง dict ซ้อนอย่างปลอดภัย
d.get("person",{}).get("name")

Part 2

การเพิ่ม แก้ไข และลบข้อมูล

Dictionary ใน Python เป็น mutable คือแก้ไขได้ตลอดเวลา หลักการง่ายๆ คือ ถ้า key ยังไม่มี = เพิ่มใหม่, ถ้ามีแล้ว = แก้ไข (ทับค่าเก่า)

การดำเนินการวิธีใช้หมายเหตุ
เพิ่ม key ใหม่d["new_key"] = valueถ้า key ไม่มีจะสร้างให้ทันที
แก้ไขค่าเดิมd["existing"] = new_valทับค่าเก่า ไม่มี error
เพิ่ม/แก้หลายคู่d.update({...})เพิ่มและแก้ไขพร้อมกัน
ลบด้วย deldel d["key"]ถ้าไม่มี key → KeyError
ลบและรับค่ากลับd.pop("key")คืนค่า value ออกมา
ลบแบบปลอดภัยd.pop("key", default)ถ้าไม่มีจะคืน default แทน
ลบรายการสุดท้ายd.popitem()Python 3.7+ ลบคู่ล่าสุด
ล้างทั้งหมดd.clear()dictionary กลายเป็น {}
