Nested Loop คืออะไร?

Nested Loop (ลูปซ้อนลูป) คือการนำ Loop หนึ่งไว้ภายใน Loop อีกอัน ใช้เมื่อต้องการทำงานกับข้อมูล 2 มิติ เช่น ตาราง (แถว × คอลัมน์) หรือต้องการเปรียบเทียบทุกคู่ข้อมูล

🔄
Outer Loop (ลูปรอบนอก)
ควบคุมจำนวนแถว หรือรอบหลัก ทำงาน 1 ครั้ง → ลูปในทำงานจนจบ
🔁
Inner Loop (ลูปรอบใน)
ควบคุมจำนวนคอลัมน์ หรือรายละเอียด ทำงานซ้ำในแต่ละรอบของลูปนอก
⏱️
จำนวนรอบรวม
= รอบลูปนอก × รอบลูปใน
เช่น range(3) × range(5) = 15 ครั้ง
📊
ความซับซ้อน O(n²)
ถ้าทั้งสองลูปมี n รอบ จะเป็น O(n²) — ระวังเมื่อข้อมูลใหญ่
โครงสร้างพื้นฐาน: for i in range(3): # ลูปนอก — i = 0, 1, 2 for j in range(5): # ลูปใน — j = 0..4 ทุกรอบ print(i, j)

Part 1

Single Loop vs Nested Loop

✅ Single Loop — O(n)
ทำงาน เชิงเส้น เหมาะกับข้อมูล 1 มิติ for i in range(5): print(i) # ทำ 5 ครั้ง ใช้เมื่อ: วนแถว, ผลรวม, ค้นหาค่า
🔷 Nested Loop — O(n×m)
ทำงาน แบบตาราง เหมาะกับข้อมูล 2 มิติ for i in range(5): for j in range(5): print(i, j) # ทำ 25 ครั้ง ใช้เมื่อ: ตาราง, Matrix, Pairwise
⚠️ ข้อควรระวัง: Nested Loop อาจช้าเมื่อข้อมูลใหญ่ — ถ้าต้องการประสิทธิภาพสูงให้พิจารณาใช้ Dictionary, Sorting, หรือ Set แทน เพื่อลดจาก O(n²) เป็น O(n)

อ่านวน Loop ทุก Cell
ใช้ for row in matrix: for val in row: อ่านค่าทุกช่องในตาราง
