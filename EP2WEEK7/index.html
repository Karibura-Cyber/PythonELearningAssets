Introduction

ทำไมต้องใช้ Function?

โจทย์ 1 ข้อ มักประกอบด้วยปัญหาย่อยหลายส่วน การเขียนทุกอย่างใน if-else ก้อนเดียว = หายนะ (Debug ยากมาก)

🔁
Loop
ใช้ทำซ้ำงานง่ายๆ ที่จบในตัว ไม่ซับซ้อน ไม่ต้องเรียกใช้จากที่อื่น
for i in range(10): print(i)
📦
Function
ใช้ห่อหุ้ม Logic ที่ซับซ้อน เช่น check_prime(n), find_gcd(a, b) เพื่อเรียกใช้ซ้ำๆ
แก้จุดเดียว เปลี่ยนทั้งโปรแกรม
ข้อดีของ Function (Modularity)
1. Reusability — เขียนครั้งเดียว เรียกใช้ได้หลายที่
2. Maintainability — แก้บั๊กที่เดียว แก้ทั้งโปรแกรม
3. Readability — โค้ดอ่านง่าย แต่ละฟังก์ชันทำงานเดียว

Part 2

Scope & Life Cycle — ตัวแปรอยู่ได้แค่ไหน?

ตัวแปรที่สร้างในฟังก์ชัน ไม่ได้อยู่ตลอดไป — มันมีขอบเขตและอายุขัยของมันเอง

🌍
Global Scope
ตัวแปรหลักของโปรแกรม ประกาศนอกฟังก์ชัน อยู่ได้ตลอด ทุกส่วนของโปรแกรมเข้าถึงได้
📍
Local Scope
ตัวแปรที่ประกาศในฟังก์ชัน "ตาย" ทันทีที่ฟังก์ชันจบ เข้าถึงจากข้างนอกไม่ได้
สิ่งที่มักเกิดขึ้น (Pitfalls):
• Shadowing — เผลอใช้ชื่อตัวแปรซ้ำกันระหว่าง local และ global
• พยายามแก้ค่า global variable โดยไม่ประกาศ global
• วิธีแก้ — ส่งค่าผ่าน Parameter เสมอ หลีกเลี่ยงการใช้ Global ถ้าไม่จำเป็น

is_prime(n) — นักตรวจสอบจำนวนเฉพาะ

เขียนฟังก์ชัน is_prime(n) รับค่าจำนวนเต็ม n Return True ถ้าเป็นจำนวนเฉพาะ, Return False ถ้าไม่ใช่

ตัวอย่าง call

ผลลัพธ์
True False True False
เขียนโค้ดex2.py
ตรวจ: ถ้า n < 2 → False. วนลูป i ตั้งแต่ 2 ถึง n-1 ถ้า n % i == 0 → False. ถ้าผ่านทุกตัว → True

เคล็ดลับเร็วกว่า: วนถึง int(n**0.5) + 1 ก็พอ

Helper Function — หาค่า Combination

4

combination(n, r) โดยใช้ factorial() เป็น Helper

สร้างฟังก์ชันลูก factorial(num) แล้วสร้างฟังก์ชันแม่ combination(n, r) โดยเรียกใช้ factorial · สูตร: n! / (r! × (n-r)!)

ตัวอย่าง call

ผลลัพธ์
10 120 1
เขียนโค้ดex4.py

แล้วใน combination(n, r) ให้ return factorial(n) // (factorial(r) * factorial(n-r))
โจทย์ที่ 5 · Basic · Recursion

การคูณแบบบวกซ้ำ (Hello Recursion)

5

multiply(a, b) — คูณโดยไม่ใช้เครื่องหมาย *

เขียนฟังก์ชัน multiply(a, b) หาผลคูณ a×b โดยใช้การบวกซ้ำด้วย Recursion · แนวคิด: a * b = a + (a * (b-1)) · Base Case: ถ้า b == 0 → return 0

ตัวอย่าง call
multiply(3, 4) multiply(5, 0) multiply(7, 1)
ผลลัพธ์
12 0 7
เขียนโค้ดex5.py
def multiply(a, b):
  if b == 0: return 0
  return a + multiply(a, b - 1)
โจทย์ที่ 6 · Mid · Recursion

หา ห.ร.ม. ด้วย Euclidean Algorithm

6

gcd(a, b) — หาตัวหารร่วมมากด้วย Recursion

Recursive Case: gcd(a, b) = gcd(b, a % b) · Base Case: ถ้า b == 0 → return a

ตัวอย่าง call
gcd(48, 18) gcd(100, 75) gcd(7, 3)
ผลลัพธ์
6 25 1
เขียนโค้ดex6.py
def gcd(a, b):
  if b == 0: return a
  return gcd(b, a % b)
โจทย์ที่ 7 · Mid · Recursion

กลับด้านข้อความ (Reverse String)

7

reverse(s) — กลับสตริงด้วย Recursion

แนวคิด: เอาตัวสุดท้ายมาตั้ง + reverse(ตัวที่เหลือ) · Base Case: ถ้าสตริงว่าง หรือมีตัวเดียว → return s

ตัวอย่าง call
reverse("HELLO") reverse("Python") reverse("A")
ผลลัพธ์
OLLEH nohtyP A
เขียนโค้ดex7.py
def reverse(s):
  if len(s) <= 1: return s
  return s[-1] + reverse(s[:-1])
โจทย์ที่ 8 · Hard · Recursion

Tower of Hanoi — หอคอยฮานอย

8

move(n, start, end, temp) — บอกขั้นตอนการย้ายจาน

มีเสา 3 ต้น จาน N ใบ · Logic: (a) ย้าย N-1 ใบ ไปเสา temp (b) ย้ายใบใหญ่สุดไปเสา end (c) ย้าย N-1 ใบ จาก temp ไป end

ตัวอย่าง call
move(2, 'A', 'C', 'B')
ผลลัพธ์
A → B A → C B → C
เขียนโค้ดex8.py
def move(n, start, end, temp):
  if n == 1:
    print(start, "→", end)
    return
  move(n-1, start, temp, end)
  print(start, "→", end)
  move(n-1, temp, end, start)
โจทย์ที่ 9 · Mid · Recursion

Fibonacci Sequence

9

fib(n) — หาลำดับ Fibonacci ด้วย Recursion

Base Case: fib(0) = 0, fib(1) = 1 · Recursive: fib(n) = fib(n-1) + fib(n-2)

ตัวอย่าง call
fib(0) fib(1) fib(6) fib(10)
ผลลัพธ์
0 1 8 55
เขียนโค้ดex9.py
def fib(n):
  if n == 0: return 0
  if n == 1: return 1
  return fib(n-1) + fib(n-2)
โจทย์ที่ 10 · Hard · Recursion

Power Function — ยกกำลัง

10

power(base, exp) — คำนวณ base^exp ด้วย Recursion (ห้ามใช้ **)

คำนวณ base ยกกำลัง exp โดยใช้ Recursion ห้ามใช้ ** หรือ pow() · แนวคิด: base^exp = base × base^(exp-1) · Base Case: exp == 0 → return 1

ตัวอย่าง call
power(2, 10) power(3, 4) power(5, 0) power(7, 1)
ผลลัพธ์
1024 81 1 7
เขียนโค้ดex10.py
def power(base, exp):
  if exp == 0: return 1
  return base * power(base, exp - 1)
